# Skill: Ângulos — 16 Categorias

> Extraído de: `skill-formula-7-passos-ads.md` (Passo 01)
> Fonte original: Agente 11 — Mestre em ADS (Squad de Copywriter)
> Adaptado para: uso em conteúdo orgânico (Social Media)

---

## Definição

**Ângulo = uma mini Big Idea.** O conteúdo tem uma Big Idea central e depois tem várias "pequenas Big Ideas" — os ângulos. Cada ângulo é um ponto de vista forte, exótico ou curioso sobre o mesmo tema, forte o suficiente para sustentar um conteúdo inteiro.

Ângulo NÃO é o mesmo que Tom Emocional. Ângulo é a **lente narrativa**; Tom Emocional é o **sentimento** que colore o conteúdo.

---

## Importância Estratégica

O ângulo é o **hack de renovação de conteúdo sem trocar o tema**. Quando um tema satura, muitos acham que esgotou. Na realidade, basta apresentar o mesmo tema com um novo ângulo — isso atrai um novo segmento de público sem mudar nada no assunto.

**Em conteúdo orgânico:** O mesmo tema pode render 5-10 conteúdos diferentes, um por ângulo. Isso multiplica a produção sem repetir.

## 16 Categorias de Ângulo

## Processo de Execução (adaptado para orgânico)

```
1. LER O BRIEFING DA PAUTA
   → Tema, emoção central, público, etapa do funil, formato
   → O briefing do Agente 03 já sugere um ângulo — usar como ponto de partida

2. EXTRAIR ELEMENTOS "ANGULIZÁVEIS"
   → Dados específicos do briefing (números, resultados, contrastes)
   → Dores/desejos do público (das Vozes do Mercado e do briefing)
   → Histórias do criador (do dossiê do Agente 01)
   → Controvérsias do nicho (da pesquisa do Agente 02)
   → Mecanismos nomeados (se existirem do Squad de Copy)
   → Elementos de autoridade do criador

3. CRUZAR COM AS 16 CATEGORIAS
   → Para cada elemento forte, testar: funciona como
     Contrarian? Como Pergunta Paradoxal? Como The ONE Thing?
   → Gerar 2-3 variações de ângulo

4. SELECIONAR O ÂNGULO MAIS FORTE
   Critérios de avaliação:
   - Especificidade: tem dados concretos?
   - Curiosidade: gera loop aberto?
   - Relevância: conecta com dor/desejo do público?
   - Diferenciação: é diferente do que os concorrentes usam?
   - Fit com formato: funciona no formato definido (carrossel, reels, thread)?

5. MONTAR O ÂNGULO
   → Essência: a ideia central em 1 frase
   → Categoria: qual das 16
   → Variações de frase: 2-3 formas de verbalizar
   → Conectar com headline (28 formas) e hook (5 fundamentos)
```

## Diferença ADS vs. Orgânico

| Aspecto | ADS | Orgânico |
|---------|-----|---------|
| Quantidade de ângulos por campanha | 2-5 por bateria de teste | 1 por conteúdo |
| Seleção | Rankear 5, testar 2 em A/B | Selecionar o melhor para aquela pauta |
| Tom | Pode ser mais direto/vendedor | Precisa parecer conteúdo, não venda |
| Renovação | Novos ângulos quando campanha satura | Novos ângulos para não repetir tema |
| Gate | Orquestrador aprova 2 para teste | Faz parte do processo de criação (sem gate separado) |

---

## Regras

1. **Nunca criar conteúdo sem definir o ângulo primeiro** — o ângulo direciona tudo (headline, hook, corpo, CTA)
2. **Nunca repetir o mesmo ângulo em conteúdos consecutivos** — diversificar
3. **O ângulo do briefing (Agente 03) é ponto de partida, não camisa de força** — o agente de plataforma pode refinar usando as 16 categorias
4. **Um mesmo tema pode render 16 conteúdos** — um por categoria de ângulo
5. **Ângulo ≠ tom emocional** — "Contrarian" é ângulo (lente narrativa), "provocativo" é tom (sentimento)
