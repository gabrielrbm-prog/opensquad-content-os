# Skill: Catálogo de Formatos, Tons Emocionais e Estratégias de Ads

> Sistema de 3 camadas para montagem de anúncios. Cada ad combina: **Formato** (como se parece) + **Tom Emocional** (qual emoção/sentimento domina) + **Estratégia** (qual tática usa) + **Receita Lego** (como a copy é montada).
>
> **Nota:** "Ângulo" neste agente refere-se à mini Big Idea narrativa do anúncio (ver Passo 01 do Framework). O que antes era chamado de "Ângulo Emocional" foi renomeado para **Tom Emocional** para evitar confusão.

---

## CAMADA 1: FORMATOS VISUAIS (como o ad se parece)

O formato é a **embalagem visual** do anúncio. Ele define a primeira impressão — antes mesmo da copy ser processada. O objetivo do formato é **não parecer anúncio**.

### Ranking de Formatos por Frequência Real (validado via análise de 21 swipe files)

| # | Formato | Frequência | Observação |
|---|---------|-----------|------------|
| 1 | **Dark Visual + Motion Graphics** | 16/16 | Presente em TODOS os ads. Números animados, setas, texto bold |
| 2 | **UGC (real e IA)** | 12/16 | Formato dominante para engajamento |
| 3 | **Split Screen / Tela Dividida** | 8/16 | SEMPRE para comparação (antes/depois, problema/solução) |
| 4 | **Expert falando direto** | 7/16 | Nunca sozinho — sempre com apoio visual |
| 5 | **Caixinha de Pergunta** | 6/16 | Formato nativo Stories/Reels, simula conteúdo orgânico |
| 6 | **Podcast / Corte de podcast** | 4/16 | Formato de autoridade, funciona para público mais velho/sofisticado |
| 7 | **Estilo documentário / história** | 4/16 | Para ads longos e high ticket |

> **Tendência confirmada:** Formato vertical 9:16 é o padrão universal. Nenhum ad do swipe foi pensado para formato horizontal.

### Diferenças de Formato por Nicho (validado via swipe)

| Aspecto | Trading/Finanças | Emagrecimento/Saúde | Creator Economy |
|---------|-----------------|---------------------|-----------------|
| Visual | Dark, azul/dourado, cinematográfico | Claro, antes/depois, UGC natural | Colorido, energético, dopaminérgico |
| Tom | Técnico → emocional | Emocional → científico | Casual, direto, aspiracional |
| Prova | Números auditados, rankings, boletas | Antes/depois, autoridade médica | Prints de pagamento, matemática simples |
| Formato dominante | POV + Dark Visual | UGC + POV narrativo | UGC + React |
| CTA | Aula/diagnóstico/grupo | Artigo/cupom/desafio | Grupo WhatsApp/vagas |

### 1A. FORMATOS NATIVOS DE VÍDEO

Já nascem como vídeo. São os formatos que o agente deve priorizar para campanhas de vídeo.

---

#### F01 — Podcast

**O que é:** Clip de conversa entre 2+ pessoas, estilo entrevista/bate-papo.

**Estrutura visual:**
- Duas ou mais pessoas sentadas frente a frente
- Microfones visíveis (reforça autenticidade)
- Cortes rápidos entre quem fala e quem reage
- Legendas na parte inferior

**Quando usar:**
- Expert com boa oratória e carisma
- Temas que precisam de explicação mais profunda
- Mercado sofisticado (nível 3-5) — parece conteúdo, não venda
- Quando o expert tem um co-host ou entrevistador

**Plataformas:** Meta (Feed, Reels), YouTube (Shorts, Pre-roll), TikTok

**Duração ideal:** 30s-2min (clip extraído de conversa maior)

**Compatibilidade com Receitas Lego:**
- Receita #1 (Educacional) — explicar mecanismo em formato conversa
- Receita #4 (Contrarian) — debater crença comum
- Receita #5 (Diferenciação) — comparar com alternativas

**Avatar ideal:** Expert + Entrevistador

---

#### F02 — UGC Testemunho

**O que é:** Pessoa comum filmando a si mesma contando sua experiência com o produto/solução. Câmera frontal, cenário casual (casa, carro, escritório).

**Estrutura visual:**
- Selfie-cam ou câmera frontal
- Cenário real (não estúdio)
- Iluminação natural
- Tom conversacional, sem script aparente

**Quando usar:**
- Público frio (nível consciência 1-3) — prova social funciona melhor que expert falando
- Mercado sofisticado — UGC parece menos "ad"
- Quando existem clientes reais com resultados documentados
- Campanhas de escala (múltiplos UGCs = múltiplos ângulos)

**Plataformas:** Meta (Reels, Stories, Feed), TikTok

**Duração ideal:** 15s-60s

**Compatibilidade com Receitas Lego:**
- Receita #2 (Emocional) — história pessoal de transformação
- Receita #3 (Prova) — resultado real demonstrado
- Receita #6 (Facilidade) — "foi mais fácil do que eu imaginava"

**Avatar ideal:** Cliente/Usuário real (ou ator simulando UGC)

---

#### F03 — GreenScreen (React)

**O que é:** Pessoa na frente de tela verde (ou efeito nativo do TikTok/Reels) reagindo a algo projetado atrás — print, gráfico, notícia, tweet, vídeo.

**Estrutura visual:**
- Pessoa no canto inferior (geralmente esquerda)
- Conteúdo projetado atrás ocupando ~70% da tela
- Pessoa aponta, reage, comenta
- Legendas na frente

**Quando usar:**
- Reagir a dados, notícias, prints que provam o ponto
- Formato muito nativo do TikTok — performa bem com público jovem
- Quando tem material visual forte para mostrar (gráficos, prints de resultado, matérias)

**Plataformas:** TikTok (nativo), Meta Reels, YouTube Shorts

**Duração ideal:** 15s-45s

**Compatibilidade com Receitas Lego:**
- Receita #3 (Prova) — reagir a dados/resultados
- Receita #4 (Contrarian) — reagir a crença comum com dados opostos
- Receita #1 (Educacional) — explicar algo mostrando visual atrás

**Avatar ideal:** Expert ou Influenciador parceiro

---

#### F04 — POV (Olhar do Prospect)

**O que é:** Câmera filmando como se estivesse nos olhos do prospect. O espectador vê exatamente o que a persona viveria ao usar a solução.

**Estrutura visual:**
- Câmera em primeira pessoa
- Mãos aparecendo na tela (como se fossem do espectador)
- Ambiente real da experiência
- Narração em off ou texto na tela

**Quando usar:**
- Produtos com experiência visual forte (abrir plataforma, entrar numa sala, ver resultado)
- Eliminar abstração — mostrar exatamente como é
- Reduzir fricção de "não sei como funciona"

**Plataformas:** TikTok, Meta Reels, YouTube Shorts

**Duração ideal:** 15s-30s

**Compatibilidade com Receitas Lego:**
- Receita #6 (Facilidade) — "olha como é simples, veja com seus olhos"
- Bloco #18 (Leitor Dentro da Ação) — encaixe perfeito com POV

**Avatar ideal:** Sem rosto (câmera é o prospect) + narração

---

#### F05 — Skincare / Atividade Paralela

**O que é:** Pessoa fazendo uma atividade cotidiana (maquiar, cozinhar, limpar, cortar) enquanto narra a copy. A atividade prende a atenção visual; a narração entrega a mensagem.

**Estrutura visual:**
- Close na atividade manual
- Pessoa falando naturalmente enquanto faz algo
- Tom de "conversa casual" — como se estivesse falando com amiga

**Quando usar:**
- Público feminino (forte afinidade com o formato)
- Produtos de saúde, beleza, bem-estar
- Quando quer mascarar que é ad — formato ultra-nativo

**Plataformas:** TikTok (muito forte), Meta Reels

**Duração ideal:** 30s-60s

**Compatibilidade com Receitas Lego:**
- Receita #2 (Emocional) — história pessoal
- Receita #6 (Facilidade) — "fiz isso enquanto me maquiava"

**Avatar ideal:** UGC (pessoa comum) ou Atriz

---

#### F06 — GRWM (Get Ready With Me)

**O que é:** "Se arrume comigo" — pessoa se arrumando (espelho, banheiro, closet) enquanto conta como resolveu um problema ou descobriu a solução.

**Estrutura visual:**
- Espelho ou câmera frontal no banheiro/closet
- Pessoa se arrumando (maquiando, vestindo, penteando)
- Narração natural, estilo diário
- Legendas na tela

**Quando usar:**
- Público feminino predominantemente
- Storytelling leve — funciona bem para história de descoberta
- Formato ultra-nativo do TikTok e Reels

**Plataformas:** TikTok, Meta Reels

**Duração ideal:** 30s-90s

**Compatibilidade com Receitas Lego:**
- Receita #2 (Emocional) — "enquanto me arrumo, deixa eu te contar..."
- Bloco #16 (História Descoberta) — narrativa de como descobriu

**Avatar ideal:** UGC ou Atriz

---

#### F07 — Entrevista na Rua

**O que é:** Entrevistador aborda pessoa na rua com pergunta provocativa ou polêmica. Formato de "conteúdo social" que gera engajamento natural.

**Estrutura visual:**
- Câmera seguindo entrevistador
- Pessoas reais na rua
- Perguntas que geram reações genuínas
- Cortes rápidos entre respostas

**Quando usar:**
- Gerar prova social espontânea
- Perguntas que revelam a dor/crença do público
- Formato viral — alto potencial de compartilhamento
- Mercados com temas que geram opinião forte

**Plataformas:** TikTok (muito forte), Meta Reels, YouTube Shorts

**Duração ideal:** 30s-60s

**Compatibilidade com Receitas Lego:**
- Receita #4 (Contrarian) — revelar que crença comum está errada
- Bloco #9 (Call Out Problem) — as próprias pessoas na rua verbalizam a dor

**Avatar ideal:** Entrevistador + Público real

---

#### F08 — Edição Motion

**O que é:** Vídeo montado com motion graphics, transições cinematográficas, textos animados. Sem necessidade de talking head — a edição É o conteúdo.

**Estrutura visual:**
- Imagens/clips de stock ou do produto
- Motion graphics (números animados, textos voando, transições)
- Narração em off (voz humana ou IA)
- Música de fundo que acompanha o ritmo da edição

**Quando usar:**
- Quando não tem expert ou UGC disponível
- Produtos complexos que precisam de demonstração visual
- Ads de escala (produção rápida, sem depender de gravação)
- Formato dark/cinematográfico

**Plataformas:** Meta (Feed, Reels), YouTube (Pre-roll, Shorts), TikTok

**Duração ideal:** 15s-90s

**Compatibilidade com Receitas Lego:**
- Receita #1 (Educacional) — explicar mecanismo com visual
- Receita #3 (Prova) — animar dados e estatísticas
- Receita #5 (Diferenciação) — split screen comparativo

**Avatar ideal:** Narração com IA ou Voz profissional

---

#### F09 — Dexter / Sgt. Doakes (Formato Viral Meme)

**O que é:** Usa template de meme viral: "Quando você sabe [algo negativo/verdade oculta] mas não pode provar." Formato meme animado com humor.

**Estrutura visual:**
- Template de meme reconhecível (Dexter, Drake, etc.)
- Texto na tela com a afirmação polêmica
- Curto e punchy
- Som/música do meme original

**Quando usar:**
- Público jovem (18-35)
- Verdades polêmicas sobre o mercado
- Formato de alto engajamento (comentários, compartilhamentos)
- TikTok e Reels como canal principal

**Plataformas:** TikTok, Meta Reels

**Duração ideal:** 5s-15s

**Compatibilidade com Receitas Lego:**
- Bloco #5 (Contrarian Nature) — verdade oposta ao senso comum
- Funciona melhor como gancho + CTA (sem corpo longo)

**Avatar ideal:** Sem avatar — formato meme puro

---

#### F10 — Hack do Corpo (Clickbait Visual)

**O que é:** Pessoa pegando algo e aplicando no corpo (esfregando, colando, bebendo). O visual incomum prende a atenção por curiosidade antes da copy começar.

**Estrutura visual:**
- Close-up na ação (mãos, corpo)
- Ação inusitada ou curiosa
- Narração explicando o "hack"
- Legendas reforçando

**Quando usar:**
- Nichos de saúde, beleza, bem-estar
- Necessidade de reter atenção nos primeiros 3 segundos
- Público frio — funciona como pattern interrupt

**Plataformas:** TikTok, Meta Reels

**Duração ideal:** 15s-45s

**Compatibilidade com Receitas Lego:**
- Bloco #4 (Sugerir Mecanismo) — "esse truque faz X"
- Receita #1 (Educacional) — explicar por que funciona

**Avatar ideal:** UGC ou Ator

---

#### F11 — Receita Estranha

**O que é:** Pessoa preparando uma receita incomum — formato tutorial culinário, mas a "receita" resolve um problema (chá para emagrecer, suco para energia, etc.).

**Estrutura visual:**
- Filmagem de cima (top-down) ou close lateral
- Ingredientes sendo adicionados
- Passo a passo visual
- Narração explicando benefícios

**Quando usar:**
- Nicho de saúde, nutrição, bem-estar
- Produtos que envolvem consumo/aplicação
- Formato altamente nativo e pouco intrusivo

**Plataformas:** TikTok, Meta Reels

**Duração ideal:** 30s-60s

**Compatibilidade com Receitas Lego:**
- Bloco #22 (Facilidade Absurda) — "é só misturar 3 ingredientes"
- Bloco #12 (Demo Visual) — mostra o produto/processo em ação

**Avatar ideal:** UGC ou Mãos na câmera (sem rosto)

---

#### F12 — Print de Vídeo com Legenda Amarela

**O que é:** Clip de vídeo (corte de podcast, live, aula) com legendas amarelas chamativas estilo "corte viral". Simula conteúdo orgânico compartilhado.

**Estrutura visual:**
- Clip vertical de vídeo existente
- Legendas grandes em amarelo (ou branco com sombra)
- Sem edição pesada — parece orgânico
- Barra de contexto no topo (opcional)

**Quando usar:**
- Reaproveitar conteúdo existente do expert
- Formato ultra-nativo — parece corte viral, não ad
- Expert com falas impactantes gravadas

**Plataformas:** Meta Reels, TikTok, YouTube Shorts

**Duração ideal:** 15s-60s

**Compatibilidade com Receitas Lego:**
- Qualquer receita — depende do conteúdo do clip
- Funciona especialmente com Bloco #5 (Contrarian) e Bloco #14 (Credibilidade)

**Avatar ideal:** Expert (clip existente)

---

#### F13 — Tentando Explicar

**O que é:** Pessoa tentando explicar um conceito de forma cômica, exagerada ou dramatizada. Formato de humor que prende pela performance.

**Estrutura visual:**
- Pessoa gesticulando muito
- Expressões exageradas
- Cortes rápidos (jump cuts)
- Tom humorístico mas com mensagem real por trás

**Quando usar:**
- Público jovem
- Conceitos que são difíceis de explicar normalmente
- Quando humor combina com o posicionamento da marca

**Plataformas:** TikTok, Meta Reels

**Duração ideal:** 15s-45s

**Compatibilidade com Receitas Lego:**
- Receita #1 (Educacional) — explicar com humor
- Bloco #25 (Metáfora Viva) — simplificar conceito complexo

**Avatar ideal:** Expert com talento cômico ou Ator

### 1B. FORMATOS ADAPTÁVEIS (nascem estáticos, podem virar vídeo)

Para cada formato, a coluna "Adaptação para Vídeo" mostra como transformar.

---

#### F14 — Formato Notícias

**Estático:** Screenshot de portal de notícias (G1, Folha, etc.) com manchete impactante.

**Adaptação vídeo:** Âncora de jornal (ator ou IA) lendo a "notícia" em formato de reportagem. OU GreenScreen reagindo à manchete.

**Quando usar:** Novidades de mercado, descobertas científicas, dados chocantes.

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo), YouTube (vídeo)

**Receitas compatíveis:** Receita #3 (Prova), Bloco #11 (Prova Numérica)

---

#### F15 — Página de Fofoca (Choquei)

**Estático:** Post visual estilo página de fofoca com manchete sensacionalista.

**Adaptação vídeo:** Pessoa reagindo à "notícia" chocante em formato GreenScreen. Ou lendo o post e comentando com expressão de choque.

**Quando usar:** Revelações sobre o mercado, verdades chocantes, exposição de práticas ruins.

**Plataformas:** Meta (estático), TikTok (vídeo), Meta Reels (vídeo)

**Receitas compatíveis:** Bloco #5 (Contrarian Nature), Receita #4 (Contrarian)

---

#### F16 — Caixinha de Pergunta

**Estático:** Screenshot do Q&A do Instagram Stories.

**Adaptação vídeo:** Mostrar a caixinha na tela e responder na câmera (estilo Stories respondendo pergunta de seguidor).

**Quando usar:** Quebrar objeções de forma natural, responder dúvidas comuns, criar familiaridade.

**Plataformas:** Meta (estático Stories, vídeo Reels), TikTok (vídeo)

**Receitas compatíveis:** Bloco #23 (Quebra Objeção), Receita #6 (Facilidade)

---

#### F17 — MEME

**Estático:** Imagem com texto no formato meme clássico (template reconhecível).

**Adaptação vídeo:** Meme animado com narração, transição, ou formato "explain the meme" onde alguém contextualiza.

**Quando usar:** Público jovem, alto engajamento, formato viral, humor.

**Plataformas:** Meta (estático Feed), TikTok (vídeo), Meta Reels (vídeo)

**Receitas compatíveis:** Funciona como gancho rápido + CTA curta

---

#### F18 — Print de Conversa

**Estático:** Screenshot de conversa WhatsApp, DM, iMessage.

**Adaptação vídeo:** Gravação de tela scrollando a conversa com narração em off ou reação da pessoa. OU "deixa eu te mostrar essa conversa..." filmando a tela do celular.

**Quando usar:** Prova social (conversa com cliente satisfeito), resultado (print de lucro), conexão (conversa relatable).

**Plataformas:** Meta (estático), TikTok (vídeo), Meta Reels (vídeo)

**Receitas compatíveis:** Receita #3 (Prova), Bloco #13 (Prova Social)

---

#### F19 — Chat GPT

**Estático:** Screenshot da conversa com ChatGPT provando um ponto.

**Adaptação vídeo:** Screen recording digitando a pergunta e mostrando o GPT respondendo em tempo real. Narração: "Olha o que o ChatGPT fala sobre..."

**Quando usar:** Usar autoridade do GPT para validar o argumento. Funciona porque público geral confia no ChatGPT.

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo)

**Receitas compatíveis:** Bloco #11 (Prova Numérica), Bloco #14 (Credibilidade Rápida)

---

#### F20 — Notificação de Mensagem

**Estático:** Mockup de notificação push no celular.

**Adaptação vídeo:** Filmagem do celular recebendo notificação + reação da pessoa. Ou mockup animado.

**Quando usar:** Criar urgência, simular alerta importante, "acabei de receber isso..."

**Plataformas:** Meta (estático), TikTok (vídeo)

**Receitas compatíveis:** Funciona como gancho de abertura (pattern interrupt)

---

#### F21 — Print de Twitter / X

**Estático:** Screenshot de tweet impactante.

**Adaptação vídeo:** Scroll de thread no Twitter com narração ou reação da pessoa no canto.

**Quando usar:** Frases de impacto, opiniões polêmicas, dados virais.

**Plataformas:** Meta (estático), TikTok (vídeo GreenScreen)

**Receitas compatíveis:** Bloco #5 (Contrarian), Bloco #14 (Credibilidade)

---

#### F22 — Change My Mind

**Estático:** Template meme "Change My Mind" com frase polêmica.

**Adaptação vídeo:** Pessoa sentada na mesa "change my mind" debatendo, ou simplesmente olhando pra câmera e defendendo a afirmação.

**Quando usar:** Posicionamento forte, opinião polêmica que defende o ponto da oferta, mercado sofisticado.

**Plataformas:** Meta (estático), TikTok (vídeo), Meta Reels

**Receitas compatíveis:** Receita #4 (Contrarian), Bloco #5 (Contrarian Nature)

---

#### F23 — Contraste (Burro x Inteligente)

**Estático:** Split image mostrando a forma errada vs. a forma certa.

**Adaptação vídeo:** Split screen com dois cenários rodando simultaneamente. OU mesma pessoa fazendo dos dois jeitos com transição.

**Quando usar:** Diferenciar método vs. alternativas, mostrar superioridade da solução.

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo)

**Receitas compatíveis:** Receita #5 (Diferenciação), Bloco #20 (Contraste Devastador)

---

#### F24 — Evolução

**Estático:** Sequência de imagens do pior ao melhor (grid ou carrossel).

**Adaptação vídeo:** Transições mostrando evolução progressiva até chegar na solução. Efeito "upgrade" visual a cada etapa.

**Quando usar:** Mostrar que sua solução é a evolução natural do mercado.

**Plataformas:** Meta (carrossel estático, vídeo), TikTok (vídeo)

**Receitas compatíveis:** Receita #5 (Diferenciação), Bloco #3 (Comparativo MDP vs MDS)

---

#### F25 — Escolha um Lado

**Estático:** Imagem dividida com opção A vs. opção B.

**Adaptação vídeo:** Apresentar as duas opções com demonstração ou argumento, pedir para o espectador "escolher". Gera engajamento nos comentários.

**Quando usar:** Criar engajamento, forçar auto-seleção, polarizar (sua solução vs. o mercado).

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo — alto engajamento)

**Receitas compatíveis:** Bloco #20 (Contraste Devastador), Receita #5 (Diferenciação)

---

#### F26 — Nós vs. Eles

**Estático:** Tabela comparativa visual (nossa solução vs. concorrência).

**Adaptação vídeo:** Split screen mostrando diferenças na prática. OU narração com lista comparativa animada.

**Quando usar:** Diferenciação direta, mercado com alternativas conhecidas.

**Plataformas:** Meta (estático e vídeo), YouTube (vídeo)

**Receitas compatíveis:** Receita #5 (Diferenciação), Bloco #20 (Contraste Devastador)

---

#### F27 — Antes & Depois

**Estático:** Split image com resultado visual (antes/depois).

**Adaptação vídeo:** Timelapse ou transição mostrando a transformação. OU pessoa mostrando o "antes" e depois a transição revela o "depois".

**Quando usar:** Produtos com resultado visual (saúde, beleza, fitness, finanças com prints).

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo)

**Receitas compatíveis:** Receita #3 (Prova), Bloco #12 (Demo Visual)

**Atenção compliance:** Meta tem restrições para antes/depois em saúde. Cuidado com políticas.

---

#### F28 — Comparação (Demonstrativa)

**Estático:** Side-by-side visual.

**Adaptação vídeo:** Demo prática comparando soluções em tempo real (ex: limpando com produto A vs. produto B).

**Quando usar:** Quando é possível demonstrar superioridade visualmente.

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo)

**Receitas compatíveis:** Bloco #12 (Demo Visual), Bloco #3 (Comparativo MDP vs MDS)

---

#### F29 — Anúncios Escritos à Mão

**Estático:** Foto de texto escrito em quadro/papel.

**Adaptação vídeo:** Filmando a pessoa escrevendo em tempo real (whiteboard, caderno, quadro) — cria curiosidade pelo que vem a seguir.

**Quando usar:** Tom pessoal e autêntico, "carta aberta", confissão, mensagem direta.

**Plataformas:** Meta (estático e vídeo), TikTok (vídeo)

**Receitas compatíveis:** Receita #2 (Emocional), Bloco #16 (História Descoberta)

---

#### F30 — Conteúdo Social (Simulação de Chat)

**Estático:** Mockup de conversa WhatsApp, DM do Instagram, post do Twitter.

**Adaptação vídeo:** Gravação de tela scrollando a conversa com narração ou música.

**Quando usar:** Prova social, criar relatabilidade, "olha essa conversa..."

**Plataformas:** Meta (estático), TikTok (vídeo)

**Receitas compatíveis:** Bloco #13 (Prova Social)

### 1C. FORMATOS ESTÁTICO-ONLY

Formatos cuja força está na escaneabilidade visual instantânea. Adaptar para vídeo dilui o impacto.

---

#### F31 — Anúncios de Manchete

**O que é:** Headline forte + sub-headline + imagem impactante + CTA. O ad estático clássico.

**Por que funciona melhor estático:** A força está no impacto visual instantâneo — o cérebro processa em <2 segundos. Vídeo adicionaria tempo desnecessário.

**Plataformas:** Meta (Feed, Stories)

---

#### F32 — Pontos de Destaque (Bullet Points)

**O que é:** Lista visual de 3-7 benefícios ou features com ícones/checkmarks.

**Por que funciona melhor estático:** Escaneabilidade. O olho percorre a lista em segundos. Em vídeo, cada bullet precisaria de tempo — perde a velocidade.

**Plataformas:** Meta (Feed, Stories)

---

#### F33 — Listagem Branca (Whitelisting)

**O que é:** Lista estilo checklist de coisas que geram o benefício desejado.

**Por que funciona melhor estático:** Mesmo princípio dos bullet points. Lista visual = escaneamento rápido.

**Plataformas:** Meta (Feed)

---

#### F34 — Estatísticas

**O que é:** Infográfico com números impactantes (porcentagens, dados, resultados).

**Por que funciona melhor estático:** Número grande + design clean = impacto instantâneo. "97,5% das pessoas..." funciona melhor como punch visual.

**Plataformas:** Meta (Feed), pode funcionar como card em carrossel

---

#### F35 — Características / Benefícios

**O que é:** Lista de features e benefícios do produto com design clean.

**Por que funciona melhor estático:** Feature list precisa de escaneabilidade para comparação mental.

**Plataformas:** Meta (Feed, Stories)

---

## CAMADA 2: TONS EMOCIONAIS (qual sentimento domina o ad)

O tom emocional é o **sentimento dominante** que permeia o anúncio. Não é um formato e não é um ângulo narrativo — é a direção emocional que colore TODA a peça.

O agente escolhe o tom emocional com base no **cluster-alvo** e no **desejo/dor dominante** desse cluster.

> **Diferença importante:**
> - **Ângulo** (Passo 01 do Framework) = mini Big Idea narrativa. É o PONTO DE VISTA sobre a oferta. Ex: "A brecha na abertura da bolsa asiática"
> - **Tom Emocional** (esta camada) = sentimento que COLORE o ad. Ex: Família, Independência, Poder
> - Um ad combina 1 ângulo + 1 tom emocional: Ângulo "brecha noturna" com tom "Independência" = "Opere 2h por noite e nunca mais dependa do seu chefe"

---

#### T01 — Família

**O que explora:** Benefício para cônjuge, filhos, pais. Conexão familiar. "Faço isso por eles."

**Dores que ataca:** Culpa por não estar presente, medo de não prover, distância emocional.

**Desejos que ativa:** Ser um provedor melhor, dar uma vida melhor, orgulho familiar.

**Clusters mais responsivos:** Pais (25-45), responsáveis por família, quem sente culpa por trabalhar demais.

**Exemplo de combinação:**
- Formato: UGC Testemunho
- Ângulo: Família
- Copy: "Minha filha me perguntou por que eu nunca tava em casa pro jantar. Naquele dia, decidi que ia encontrar outra forma de ganhar dinheiro..."

---

#### T02 — Independência

**O que explora:** Liberdade financeira, não depender de emprego, chefe, ou sistema.

**Dores que ataca:** Sensação de estar preso, dependência do salário, medo de demissão.

**Desejos que ativa:** Ser dono do próprio tempo, trabalhar de onde quiser, não prestar contas.

**Clusters mais responsivos:** CLT frustrados, autônomos buscando escala, jovens que querem alternativa ao emprego tradicional.

**Exemplo de combinação:**
- Formato: Podcast
- Ângulo: Independência
- Copy: "O dia que larguei o emprego não foi coragem. Foi que não aguentava mais depender de alguém pra decidir quanto eu ganho..."

---

#### T03 — Poder

**O que explora:** Sucesso, força, conquista, domínio. Sentir-se capaz e poderoso.

**Dores que ataca:** Impotência, fracasso repetido, sensação de ser inferior.

**Desejos que ativa:** Ser respeitado, ter controle, demonstrar competência.

**Clusters mais responsivos:** Homens 20-40, empreendedores, competitivos.

**Exemplo de combinação:**
- Formato: Edição Motion
- Ângulo: Poder
- Copy: Montagem de conquistas + narração: "Enquanto 90% fica tentando... os outros 10% já estão operando..."

---

#### T04 — Romance / Atração

**O que explora:** Ser mais atraente, impressionar parceiro(a), conquistar.

**Dores que ataca:** Insegurança com aparência, rejeição, baixa autoestima.

**Desejos que ativa:** Ser desejado(a), causar inveja, receber elogios.

**Clusters mais responsivos:** Público de beleza, saúde, lifestyle.

**Exemplo de combinação:**
- Formato: GRWM
- Ângulo: Romance
- Copy: "Meu marido olhou pra mim diferente quando comecei a [resultado]. Ele não sabe o que mudou, mas eu sei..."

---

#### T05 — Status Social

**O que explora:** Lifestyle, conquistas visíveis, reconhecimento social.

**Dores que ataca:** Vergonha da situação atual, inveja dos outros, não ser notado.

**Desejos que ativa:** Flexar conquistas, ser admirado, demonstrar sucesso.

**Clusters mais responsivos:** Jovens aspiracionais, quem busca validação social.

**Exemplo de combinação:**
- Formato: POV
- Ângulo: Status Social
- Copy: POV mostrando lifestyle (carro, viagem, restaurante) + narração sutil sobre como chegou lá

---

#### T06 — Tranquilidade

**O que explora:** Paz, ausência de estresse, rotina leve, sono tranquilo.

**Dores que ataca:** Ansiedade, noites sem dormir, estresse crônico.

**Desejos que ativa:** Acordar sem preocupação, ter rotina simples, viver sem pressão.

**Clusters mais responsivos:** Pessoas esgotadas, workaholics, quem já tentou muitas coisas e quer simplicidade.

**Exemplo de combinação:**
- Formato: UGC
- Ângulo: Tranquilidade
- Copy: "A melhor parte não é o dinheiro. É dormir tranquilo sabendo que amanhã vai funcionar de novo..."

---

#### T07 — Aceitação

**O que explora:** Conseguir fazer/ter/usar algo que antes não conseguia.

**Dores que ataca:** Limitação, exclusão, "isso não é pra mim".

**Desejos que ativa:** Pertencimento, superação, "agora eu consigo".

**Clusters mais responsivos:** Iniciantes, pessoas que já falharam antes, quem tem síndrome do impostor.

**Exemplo de combinação:**
- Formato: Antes & Depois (vídeo)
- Ângulo: Aceitação
- Copy: Transição mostrando o momento de "agora consigo" — vestir a roupa, ver o resultado, entrar na plataforma

---

#### T08 — Curiosidade

**O que explora:** Mistério, segredo, informação oculta. O cérebro não resiste a lacunas de informação.

**Dores que ataca:** Nenhuma dor específica — ataca a necessidade humana de fechar loops abertos.

**Desejos que ativa:** "Preciso saber o que é isso."

**Clusters mais responsivos:** Todos — curiosidade é universal. Funciona melhor com público frio.

**Exemplo de combinação:**
- Formato: GreenScreen
- Ângulo: Curiosidade
- Copy: "Descobri uma coisa sobre [tema] que me deixou sem dormir. Olha isso..." [mostra print/dado]

---

#### T09 — Tabus

**O que explora:** Assuntos delicados, constrangedores ou polêmicos que ninguém fala abertamente.

**Dores que ataca:** Vergonha, isolamento por não poder falar do problema.

**Desejos que ativa:** "Finalmente alguém falando sobre isso."

**Clusters mais responsivos:** Nichos de saúde íntima, problemas financeiros (dívidas), vícios, relacionamentos.

**Exemplo de combinação:**
- Formato: UGC (baixa produção, íntimo)
- Ângulo: Tabu
- Copy: "Ninguém fala sobre isso, mas... [confissão sobre o problema que a solução resolve]"

---

## CAMADA 3: ESTRATÉGIAS DE CONTEÚDO (qual tática)

A estratégia é a **abordagem tática** do ad. Define COMO a mensagem é empacotada em termos de lógica persuasiva. Pode ser combinada com qualquer formato + ângulo.

---

#### E01 — Timing (Surfar o Momento)

**O que é:** Conectar o ad a um assunto trending — notícia, evento, polêmica, meme viral do momento.

**Como usar:** Monitorar Google Trends, Twitter/X trending, e adaptar o gancho para conectar com o assunto do momento.

**Exemplo:** Se sair notícia sobre demissões em massa → gancho: "Com X mil pessoas sendo demitidas, nunca foi tão importante ter uma segunda fonte de renda..."

**Validade:** Curta — 1 a 7 dias. Produzir e publicar rápido.

---

#### E02 — Novidade IA

**O que é:** Usar o hype de inteligência artificial como gancho ou formato.

**Como usar:** "Essa IA faz X", "Usei IA para Y", "O ChatGPT confirmou que Z".

**Exemplo:** "Perguntei pro ChatGPT qual o melhor horário pra operar e olha o que ele respondeu..." [screen record]

**Validade:** Média — enquanto IA for trending topic (2024-2026+).

---

#### E03 — Marketing Negativo

**O que é:** Ensinar o que NÃO fazer, criticar solução ruim do mercado, expor práticas questionáveis.

**Como usar:** Posicionar-se CONTRA algo para se posicionar A FAVOR de algo melhor. "Pare de fazer X" → "Faça Y em vez disso."

**Exemplo:** "3 sinais de que seu curso de trading é uma armadilha: 1) Não mostra histórico real..."

**Receitas compatíveis:** Receita #4 (Contrarian), Bloco #5 (Contrarian Nature)

---

#### E04 — Mito (Myth-Busting)

**O que é:** Lista de mitos do nicho que são quebrados um a um, revelando a verdade.

**Como usar:** "Mito: [crença comum]. Verdade: [realidade]." Repetir 3-5x.

**Exemplo:** "Mito: Operar à noite não tem liquidez. Verdade: O spread às 21h é 40% menor que às 14h."

**Receitas compatíveis:** Receita #4 (Contrarian), Bloco #5 (Contrarian Nature)

---

#### E05 — Educacional

**O que é:** Ensinar algo genuinamente útil, com CTA sutil no final. O ad entrega valor real.

**Como usar:** Dar 80% de valor educacional + 20% de direcionamento para a solução. "Aprendi isso em X anos, vou te ensinar em 60 segundos."

**Exemplo:** "3 coisas que todo trader precisa saber antes de operar à noite: 1) O spread muda..."

**Receitas compatíveis:** Receita #1 (Educacional), Blocos de Mecanismo (#1-#5)

---

#### E06 — Superação de Objeções

**O que é:** Ad focado 100% em antecipar e destruir objeções. Cada bloco do ad ataca uma objeção diferente.

**Como usar:** Listar 3-5 objeções principais do público e resolver cada uma em 5-10 segundos.

**Exemplo:** "Não tenho tempo → opera 90min à noite. Não tenho dinheiro → funciona com $50. Não sei nada → sala ao vivo ensina."

**Receitas compatíveis:** Receita #6 (Facilidade), Bloco #23 (Quebra Objeção)

---

#### E07 — Problema/Solução (Arco Narrativo)

**O que é:** Começar com gancho de dor/problema e ao longo do ad apresentar a solução. Arco clássico de copywriting.

**Como usar:** Dor → Agitação → Solução → Prova → CTA.

**Exemplo:** "Você acorda 6h, trabalha 10h, chega em casa morto... [dor]. Mas e se existisse uma forma de ganhar dinheiro nas 2h antes de dormir? [solução]"

**Receitas compatíveis:** Receita #2 (Emocional), Blocos #8-#10 (Agitação, Call Out, Identificação)

---

#### E08 — História Pessoal / do Fundador

**O que é:** Storytelling da jornada do expert ou fundador. Como descobriu a solução, por que criou o produto.

**Como usar:** Estrutura: Crise → Descoberta → Resultado → Convite. Tom confessional e humano.

**Exemplo:** "2024, minha empresa entrou em crise. Só sobrava 21h-23h pra operar. Foi quando descobri que o horário 'errado' era o único horário certo..."

**Receitas compatíveis:** Bloco #16 (História Descoberta), Receita #2 (Emocional)

---

#### E09 — Celebridade / Autoridade Externa

**O que é:** Vincular celebridade, especialista famoso, ou marca conhecida ao contexto do produto.

**Como usar:** "Elon Musk disse que...", "Segundo pesquisa de Harvard...", "A Forbes publicou que..."

**Exemplo:** Mostrar clip de celebridade falando algo alinhado com o argumento do ad + transição para CTA.

**Receitas compatíveis:** Bloco #14 (Credibilidade Rápida), Bloco #11 (Prova Numérica)

---

#### E10 — Imprensa / Notícia

**O que é:** Layout de matéria jornalística — como se uma revista ou portal tivesse publicado sobre o tema.

**Como usar:** Criar visual de notícia (headline de jornal, layout de revista) com o argumento do ad. Ou usar matéria real se existir.

**Exemplo:** Print estilizado: "NOVO ESTUDO REVELA: Traders que operam à noite têm 40% mais assertividade..."

**Receitas compatíveis:** Bloco #11 (Prova Numérica), Bloco #14 (Credibilidade)

---

#### E11 — "Eu faço isso para..." (Texto-Gatilho)

**O que é:** Frase-gatilho como overlay em vídeo ou texto em estático. Cria curiosidade + identificação.

**Como usar:** "Eu faço isso para [resultado aspiracional]" + demonstração visual. Funciona como gancho.

**Exemplo:** "Eu faço isso toda noite pra ganhar em dólar enquanto todo mundo dorme" + POV abrindo notebook

---

#### E12 — "Ele/ela ainda não sabe, mas..." (Texto-Gatilho)

**O que é:** Frase-gatilho que cria suspense e curiosidade. Implica que algo está prestes a acontecer.

**Como usar:** "Ele ainda não sabe, mas [revelação sobre como a vida vai mudar]" + visual complementar.

**Exemplo:** "Ele ainda não sabe, mas daqui 30 dias vai ter sua primeira renda em dólar" + vídeo da pessoa na rotina normal

---

#### E13 — Análise / Review

**O que é:** Usuário compartilhando como foi a experiência usando o produto. Tom de avaliação honesta.

**Como usar:** "Testei [produto] por X dias. Aqui está meu veredito honesto..." — Tom de review, não de venda.

**Exemplo:** "Testei o Clube GN50 por 14 dias. Vou ser 100% honesto: nos primeiros 3 dias... [jornada]"

**Receitas compatíveis:** Receita #3 (Prova), Bloco #13 (Prova Social)

---

#### E14 — Prova Social (Compilação)

**O que é:** Compilação de depoimentos, prints de resultado, números de movimento.

**Como usar:** Juntar múltiplas provas em sequência rápida. Quantidade gera peso.

**Exemplo:** 5 prints de resultado em sequência rápida (2s cada) + narração: "Mais de 1.000 membros. Resultados reais. Públicos."

**Receitas compatíveis:** Receita #3 (Prova), Blocos #11-#14 (todos de Prova)

---

#### E15 — Depoimentos (Fragmentos)

**O que é:** Frases curtas, prints, ou trechos rápidos de depoimentos. Não é UGC completo — são fragmentos compilados.

**Como usar:** Cortes rápidos de 3-5 segundos cada, um após o outro, de diferentes pessoas falando sobre o resultado.

**Exemplo:** Pessoa 1: "Fiz R$3.200 no primeiro mês" (3s) → Pessoa 2: "Nunca pensei que fosse tão simples" (3s) → Pessoa 3: "Aprovei na mesa em 2 semanas" (3s)

**Receitas compatíveis:** Bloco #13 (Prova Social)

---

## COMO USAR ESTE CATÁLOGO (Guia Rápido para o Agente)

### Processo de Montagem de um Ad:

```
1. DEFINIR CLUSTER-ALVO
   → Qual cluster da campanha este ad ataca?

2. DEFINIR ÂNGULO NARRATIVO (Passo 01 do Framework)
   → Qual mini Big Idea vai guiar este ad?
   → O orquestrador já definiu: usar o ângulo aprovado

3. ESCOLHER TOM EMOCIONAL (Camada 2)
   → Qual sentimento domina este ad? (Família? Independência? Poder?)
   → Baseado no cluster-alvo e no desejo/dor dominante

4. ESCOLHER ESTRATÉGIA (Camada 3)
   → Qual tática serve o objetivo? (Educar? Provar? Quebrar crença?)

5. ESCOLHER FORMATO VISUAL (Camada 1)
   → Qual plataforma?
   → Qual formato é mais nativo dessa plataforma?
   → Qual avatar está disponível?

6. MONTAR COPY COM RECEITA LEGO
   → Selecionar receita base do Sistema Lego
   → Adaptar blocos ao formato e duração
   → Escrever gancho (Ângulo + Benefício/Dor)
   → Montar corpo (blocos Lego selecionados)
   → Escrever CTA

7. BRIEFING DE EDIÇÃO
   → Estrutura Invisível: propósito de cada frase
   → Direção visual: o que aparece na tela a cada momento
   → Direção de edição: cortes, transições, legendas, música
```

### Regras de Combinação:

1. **1 ad = 1 formato + 1 ângulo narrativo + 1 tom emocional + 1 estratégia + 1 receita Lego**
2. **Não misturar mais de 1 tom emocional por ad** — perde foco (pode ter tom secundário sutil, mas 1 dominante)
3. **Formato deve ser nativo da plataforma** — TikTok pede formato TikTok, não YouTube
4. **Testar variações mudando UMA camada por vez** — trocar formato com mesma copy = teste de embalagem; trocar ângulo com mesmo formato = teste de mensagem; trocar tom emocional com mesmo ângulo = teste de emoção
5. **Receita Lego define a estrutura da copy** — os blocos dentro da receita podem ser substituídos (ver regras do Sistema Lego)
6. **Ângulo é definido pelo orquestrador** — o agente rankeia os 5 mais fortes, orquestrador escolhe 2 para teste A/B. Novas remessas podem usar novos ângulos para renovar alcance
