# Prompt Engineering para Cenários de Vídeo Clone

## Quando usar
- Precisar de background customizado para o HeyGen
- Gerar B-roll temático via Nano Banana
- Criar cenários específicos para cada formato de Reels

## Estrutura do Prompt para Background
```
[STYLE] + [SETTING] + [LIGHTING] + [CAMERA] + [MOOD] + [ASPECT RATIO]
```

## Regras
- Sempre especificar 9:16 vertical para Reels/TikTok
- Incluir "No text, no watermarks" no final
- Especificar camera + lens para realismo
- Para backgrounds de clone: foco no ambiente, não em pessoas
