# Thumbnail Generator — Criação de Capas para Reels

## Objetivo
Criar thumbnails (capas de Reel) que maximizem o CTR (Click-Through Rate) no grid do perfil e na aba Reels do Instagram.

## Especificações Técnicas

### Dimensões e Formato
- **Resolução:** 1080 x 1920 pixels (9:16 vertical)
- **Área segura para texto:** 1080 x 1350 (centro do frame, fora de cortes do grid)
- **Formato de exportação:** PNG ou JPG de alta qualidade
- **Peso máximo:** 5MB

### Grid do Instagram
- No grid, o Reel aparece como 1080 x 1350 (4:5) — o topo e a base são cortados
- **REGRA:** Todo texto e elemento importante deve estar na zona central (entre y=285 e y=1635)

## 3 Estilos de Thumbnail

### Estilo 1: Text-Heavy (Texto Dominante)
**Quando usar:** Conteúdo educacional, listas, dicas
- Fundo escuro ou gradiente
- Texto em 2-3 linhas, fonte bold (mínimo 72px)
- Cores: branco/amarelo sobre fundo escuro
- Sem foto ou foto muito desfocada atrás
- Exemplo: "3 ERROS que QUEBRAM sua PROP FIRM"

**Regras:**
- Máximo 8 palavras
- Palavras-chave em CAPS ou cor diferente
- Fonte: Inter Bold, Montserrat Black, ou similar sans-serif

### Estilo 2: Face-Focus (Rosto em Destaque)
**Quando usar:** Storytelling, mindset, opinião, resultados
- Frame do vídeo com o rosto do Gabriel em destaque
- Expressão facial marcante (surpresa, seriedade, empolgação)
- Texto curto sobreposto (máximo 4 palavras)
- Borda ou moldura para destacar do feed

**Regras:**
- Rosto deve ocupar pelo menos 40% do frame
- Olhos visíveis e nítidos
- Expressão deve combinar com o conteúdo (não sorrir em vídeo sério)
- Contraste alto entre texto e fundo

### Estilo 3: Action-Shot (Ação/Gráfico)
**Quando usar:** Análise técnica, trades, notícias
- Screenshot de gráfico/trade como fundo
- Setas, círculos ou marcações visuais
- Texto indicando o resultado ou a análise
- Estilo "breaking news" para conteúdo de notícia

**Regras:**
- Gráfico deve ser legível mesmo em miniatura
- Usar cores fortes para marcações (vermelho, verde, amarelo)
- Não colocar gráfico inteiro — cortar apenas a parte relevante
- Adicionar logo @rubimfx discretamente

## Processo de Criação

### Passo 1: Selecionar Frame de Impacto
- Assistir o vídeo final e pausar nos momentos mais expressivos
- Priorizar frames onde o Gabriel tem expressão facial marcante
- Para vídeos com gráfico, capturar o momento do "reveal"

### Passo 2: Escolher Estilo
Baseado no formato do vídeo:
| Formato do Vídeo | Estilo Recomendado |
|-------------------|-------------------|
| Revelation Hook | Text-Heavy |
| Mindset Confession | Face-Focus |
| Chart Explanation | Action-Shot |
| News + Analysis | Action-Shot ou Text-Heavy |
| Proof Reel | Face-Focus |
| Ranking/List | Text-Heavy |
| Clip Drop | Face-Focus |
| Before/After | Action-Shot |
| 3-Second Rule | Text-Heavy |
| Student Result | Face-Focus |

### Passo 3: Adicionar Texto
- Extrair a frase-chave do hook
- Condensar para máximo 8 palavras
- Aplicar hierarquia visual (palavra principal maior)
- Garantir legibilidade em miniatura (testar em 150x150px)

### Passo 4: Ajustes Finais
- Brilho e contraste adequados
- Verificar legibilidade no grid (zoom out)
- Garantir que não conflita com UI do Instagram (ícones de play, duração)
- Salvar em alta resolução

## Ferramentas

### Nano Banana (IA)
- Gerar thumbnails customizadas quando nenhum frame do vídeo funciona
- Prompt: descrever a cena desejada + texto + estilo
- Pós-processar: adicionar texto com Canva ou Figma

### Canva
- Templates prontos para thumbnails de Reels
- Fácil para texto + foto + elementos gráficos

### Figma
- Para thumbnails mais elaboradas
- Manter biblioteca de componentes reutilizáveis

## Checklist de Qualidade

- [ ] Texto legível em miniatura (150x150px)
- [ ] Rosto visível e expressivo (se estilo Face-Focus)
- [ ] Contraste alto entre texto e fundo
- [ ] Dentro da zona segura (sem cortes no grid)
- [ ] Não conflita com ícones do Instagram
- [ ] Consistente com a estética do perfil @rubimfx
- [ ] Frase de impacto (não genérica)
- [ ] Resolução 1080x1920

## Anti-Padrões (EVITAR)

- Texto pequeno demais (< 48px)
- Muitas palavras (> 10)
- Fundo poluído (gráfico inteiro sem destaque)
- Cores apagadas/sem contraste
- Foto genérica de banco de imagens
- Mesmo layout para todos os vídeos (variação é essencial)
- Clickbait que não entrega (promessa ≠ conteúdo)
