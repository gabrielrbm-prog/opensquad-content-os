## Como Usar Este Banco de Dados

### Para o Rex Roteirista:
1. Receber o tema e formato do Cal Calendar
2. Consultar esta tabela pela categoria que mais combina
3. Escolher 2 hooks (A e B) para teste A/B quando aplicável
4. Adaptar o hook ao conteúdo específico (trocar valores, ativos, etc.)

### Para A/B Testing:
1. Selecionar 2 hooks de categorias DIFERENTES para o mesmo conteúdo
2. Exemplo: Loss Aversion (#3) vs Curiosity Gap (#22) para vídeo sobre stop loss
3. Documentar qual performou melhor

### Regras:
- Hooks devem ser FALADOS + exibidos como TEXTO NA TELA
- Adaptar o hook ao conteúdo (não usar literalmente se não encaixar)
- Não repetir o mesmo hook em menos de 30 dias
- Marcar hooks já usados com data de uso
- Adicionar novos hooks conforme descobertos (meta: 200+ até fim do trimestre)
