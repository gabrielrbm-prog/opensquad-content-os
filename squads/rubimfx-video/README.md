# rubimfx-video — Squad de Produção de Vídeos com Clone IA

Squad para produção automatizada de vídeos curtos (Reels/TikTok) usando clone de voz e imagem.

## Stack Tecnológica
| Ferramenta | Função | Custo |
|---|---|---|
| HeyGen Avatar IV | Clone facial (vídeo) | $29-149/mês |
| ElevenLabs | Clone de voz PT-BR | $22/mês |
| Submagic Pro | Legendas animadas | $23/mês |
| Pexels API | B-roll grátis | $0 |
| Ayrshare/Playwright | Publicação | $0-99/mês |

## Agentes
- **Rex Roteirista** — Cria roteiros virais (10 formatos)
- **Hugo HeyGen** — Gera vídeo do clone
- **Elia ElevenLabs** — Clona a voz em PT-BR
- **Sam Submagic** — Legendas e edição
- **Bea B-Roll** — Curadoria de vídeos/imagens de apoio
- **Pablo Publisher** — Publicação Instagram + TikTok
- **Tina Trend** — Pesquisa tendências virais

## Como Usar
```bash
/opensquad run rubimfx-video
```

## Formatos de Vídeo
1. Clip Drop (corte de mentoria)
2. Revelation Hook (declaração contrária)
3. Chart Explanation (tela + voiceover)
4. Proof Reel (resultado prop firm)
5. 3-Second Rule (insight ultra-curto)
6. News + Analysis (macro do dia)
7. Mindset Confession (storytelling)
8. Before/After Trade (replay)
9. Ranking/List (top 3/5)
10. Student Result (prova social)
