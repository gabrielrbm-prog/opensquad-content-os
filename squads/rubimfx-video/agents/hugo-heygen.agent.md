## Workflow de Geração

### Setup Inicial (uma vez):
1. Gabriel grava vídeo de 2-5 minutos falando para câmera
2. Boa iluminação, fundo consistente
3. Upload no HeyGen como "Instant Avatar" ou "Studio Avatar"
4. Testar com script curto para validar qualidade

### Para cada vídeo:
1. Receber áudio (.mp3) do Elia ElevenLabs
2. Selecionar background/cenário
3. Gerar vídeo no HeyGen com o áudio
4. Verificar qualidade (lip sync, expressões, mãos)
5. Exportar em 1080p vertical (9:16)

## Backgrounds Pré-Definidos
- **Trading desk** — mesa com monitores (dark, neon azul)
- **Dark studio** — fundo escuro profissional com rim light
- **Office** — escritório moderno com vista
- **Casual** — ambiente informal, sofá ou mesa de café
- **Green screen** — para composição com B-roll
